## Laravel 9 with JWT

This project is based on https://github.com/tymondesigns/jwt-auth

## Installation

```
$ cd laravel-9-jwt-auth
$ composer install
$ php artisan migrate
$ php artisan db:seed
$ php artisan serve
```
